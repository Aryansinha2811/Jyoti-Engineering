# Jyoti Engineering Web Platform

A high-performance, full-stack web application developed for **Jyoti Engineering**[cite: 1]. This platform bridges an impactful, mobile-responsive digital storefront for industrial clients with a secure, cloud-managed administrative backend for real-time content operations[cite: 1].

Built as a tailored solution to help scale their digital presence and streamline how they showcase their core services[cite: 1].

## 🚀 Key Features

*   **Dynamic Public Storefront:** A seamless, single-page customer journey featuring a responsive service catalog, "Why Choose Us" trust grid, and an interactive lead-capture form routed directly to corporate email[cite: 1].
*   **Secured Admin Portal:** An isolated `/admin` gate with server-side validation to restrict access to authorized internal team members[cite: 1].
*   **Central Operations Control:** An intuitive dashboard for the team to instantly add or remove showcased services without touching a line of code[cite: 1].
*   **Automated Image Optimizer:** Automatically scales and converts massive field images down to ultra-lightweight formats before storing them to protect page loading speeds[cite: 1].

## 🛠️ Tech Stack

*   **Core Architecture:** Next.js, React, TypeScript[cite: 1]
*   **Styling & UI:** Tailwind CSS (Optimized for mobile, tablet, and desktop monitors)[cite: 1]
*   **Database & Storage:** Supabase Cloud Database & Storage[cite: 1]
*   **Communication:** Secure server-side email protocols routed directly to corporate endpoints[cite: 1]
